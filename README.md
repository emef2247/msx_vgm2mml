# MSX VGM to MML Converter

This script generates pass3.mml outputs from existing *_log.scc.csv and *_log.psg.csv inputs.

## Usage

To use the script, you can execute one of the following commands:

```bash
python msx_vgm2mml.py scc <log_file>
python msx_vgm2mml.py psg <log_file>
```

Make sure to have the required inputs in the appropriate format.
